# Sticky Notes

A light weight Sticky Notes Library which you can just plug into your web pages.

Fully Customizable and easy to modify to fit your needs

## Version

1.1.1

## License

MIT Licence

### Dependencies
- [jquery.js](https://code.jquery.com/jquery-3.3.1.min.js)

### Basic usage

- JavaScript

```javascript
    $("body").sticky({
		right : true,
		bottom : true
	});
```
